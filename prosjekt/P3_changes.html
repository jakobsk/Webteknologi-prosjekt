Content

  1. "Front page - front_page.html" The front page is the landing page of the website. It will contain selected offers that will be selected randomly each time a user loads the front page. The random selection will be done with javascript
  2. "About" This section will describe the business concept of XXXL and who the persons behind the concept are. It will have a picture of the creators of the webstore and some text about their background. It will also describe why XXXL is the best alternative when shopping sports equipment online.
  3. "Categories"
    1. "Football"
    2. "Running"
    3. "Tennis"
    This section will give you three alternatives when clicking on it. When clicking one of the three alternatives the user will go to a page that shows pictures of available sports equipment in each category. It will have the price and a brief decscription of each item. When clicking a picture the picture will expand it it will display more information. This will be done using javasrcipt. If a user clicks on "add to cart" the item will be put in the cart. This will also be done with javascript
  4. "Contact us" This page will contain an e-mail for customer support.
  5. "Shopping Cart" This will show all the items the user has added to the shopping cart in that session, and the total price of all items in the shopping cart

Minimum Requirements

The website will contain five javascript-enabled functionailies to create a more dynamic and responsive website. We image the following scripts to work this way:

Torgeir and Audun will be responsible for the development of this script:

  • image_script.js. This script is responsible for enlarging a picture if a user clicks on it. Furthermore, there will also be some additional text displayed when this function is invoked. The text will contain a short description of some features of the respective product.
  • shopping_cart.js This script will dynamically update the shopping cart with the item the user wants. It will also contain code to remove certain items if demanded.

Øystein and Per will be responsible for the development of this script:

  • random_offers.js A javascript function that determines which items that are displayed on the front page, as well as which that are on sale at the moment.
  • autofill_content Will be responsible for creating more html content on a website. That is, we don't want to write all the html code ourself (which would be highly repatative work), but rather let javascript do this for us. We picture us that this function will provide all html pages with content

Jakob will be responsible for the development of this script:

  • form_validation In the contact us page, we envision a html form capability such that users could send us their information. If we enable user input, we must ensure input-validation so that the input conforms with the intended format.
